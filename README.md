# TripInKorea
***
## 국내여행지 소개사이트!!
### 여행을 사진과 함께 기록하고 공유하는 웹 어플리케이션

<br>

### 프로젝트 선정 이유
- 현 시국으로 인해 여행을 다니지 못하는 사람들에게는 간접 체험을 기회를 제공하고 해외여행이 부담스러운 사람들에게는 다른 사람이 경험한 좋은 국내여행지를 둘러볼 수 있는 사이트
- 간단한 이미지와 짧은 글로 여행지에 대한 구체적인 정보전달보다 인사이트를 주기 위함
